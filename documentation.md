## Примеры синтаксиса

### Линки

#### Link - Используется для слоёв
```
[90] -> [10; sigmoid]
```

#### ReverseLink - Используется для присваивания

##### Присваивание слоя переменной
```
c <- [60; sigmoid]
```

##### Считывание данных из файла
``` 
a <- "input_a.txt"
b <- "input_b.txt"
space <- "c.txt"
```

### Слои:

#### Обычный скрытые слои

##### Создание сети цепочкой
```
[70] -> [10; sin]
```

##### То же, что и выше, но с использование переменных
```
a <- [70]
a -> [10; sin]
```

##### Другие примеры:
```
basis_1 <- {10, 20, 30, 4, 5, 6, 7}
start <- [70]
hidden <- [7; sigmoid; basis_1]
end <- [10; cos]

start -> hidden -> end
```


### Свободные члены
```
# Числа
bias_1 <- {10, 20, 30, 50, 60}

[80] -> [20; cos; bias_1] -> [30]
```

### Свёрточные слои

#### Создание ядра
```
kernel_1 <- (3; 3; sigmoid)
```

#### Создание свёрточной сети

```
[90] -> [30; func_2] -> (3; 3; func_1) -> [20]
```
##### Примечение. Ядро образует новый рассчитанный слой. То есть после применения ядра, будет создана матрица весов, которая дальше будет применяться
